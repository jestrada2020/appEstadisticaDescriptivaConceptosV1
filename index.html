Polígonos de Frecuencias Acumuladas (Ojivas)

¿Qué son las Ojivas?

Las ojivas son gráficos de línea que representan las frecuencias acumuladas. Existen dos tipos principales:

📊 Frecuencias Relativas Acumuladas

Muestra la proporción acumulada de datos hasta cada valor. Los valores van de 0 a 1 (0% a 100%).

📈 Frecuencias Absolutas Acumuladas

Muestra el número acumulado de observaciones hasta cada valor. Los valores van de 0 al total de datos (n).

Frecuencias Relativas Acumuladas

Eje izquierdo: valores decimales (0-1) | Eje derecho: porcentajes (0%-100%)

Frecuencias Absolutas Acumuladas

Eje izquierdo: número de datos | Eje derecho: porcentajes (0%-100%)

Casos Especiales: Datos Pares vs Impares

📊 Número Par de Datos (n es par):

Cuando n es par, las posiciones n/4, n/2, y 3n/4 pueden no ser números enteros. En este caso, tomamos la media aritmética de los dos valores adyacentes.

Ejemplo: Si n=20, entonces n/2=10, tomamos (valor₁₀ + valor₁₁)/2

📊 Número Impar de Datos (n es impar):

Cuando n es impar, las posiciones pueden dar números enteros o decimales. Si es decimal, redondeamos hacia arriba para encontrar la posición.

Ejemplo: Si n=15, entonces n/2=7.5, tomamos el valor en la posición 8

Medidas de Dispersión

¿Qué es la Dispersión?

Las medidas de dispersión nos indican qué tan separados están los datos respecto a la media aritmética. Una medida de dispersión pequeña indica que los datos se acumulan con proximidad alrededor de la media aritmética. Por consiguiente, la media se considera representativa de los datos.

Por el contrario, una medida grande de dispersión indica que la media no es confiable.

• Dispersión Absoluta

Entre ellas el rango, el rango intercuartílico, la desviación media, la varianza y la desviación típica.

Para datos no agrupados:

DM = Σ|xᵢ - x̄| / n = Σ|dᵢ| / n

Donde dᵢ = xᵢ - x̄ (desviación de cada valor)

Para datos agrupados:

DM = Σ|xᵢₘ - x̄|fᵢ / n = Σxᵢ|dᵢ| / n

Donde xᵢₘ es la marca de clase y fᵢ la frecuencia

• Dispersión Relativa

La que tiene mayor importancia es el coeficiente de variación.

Coeficiente de Variación (CV):

CV = (S / x̄) × 100%

El coeficiente de variación (o coeficiente de Pearson) CV, se define por el cociente entre la desviación típica (o estándar) y la media aritmética de la variable e indica, por tanto, el número de veces que la desviación típica contiene a la media.

CV < 15%
Baja dispersión
Media muy representativa
15% ≤ CV ≤ 35%
Dispersión moderada
Media aceptablemente representativa
CV > 35%
Alta dispersión
Media poco representativa

Consideraciones Importantes

  • • CV = 0: Todos los valores son iguales a la media (dispersión nula)
  • • CV muy bajo: La media es perfecta representación de los datos
  • • CV muy alto: Los datos están muy dispersos, la media es poco representativa
  • • Problema del denominador cero: Si la media es 0, el CV carece de significado

Cálculo Correcto de los Bigotes:

Regla importante: Los bigotes NO se extienden hasta las barreras teóricas (Q₁ - 1.5×RI y Q₃ + 1.5×RI).

Bigote inferior: Se extiende desde Q₁ hasta el valor real más pequeño que esté dentro del rango normal (≥ Q₁ - 1.5×RI).

Bigote superior: Se extiende desde Q₃ hasta el valor real más grande que esté dentro del rango normal (≤ Q₃ + 1.5×RI).

Si existen outliers, los bigotes terminan en el último valor normal, no en la barrera teórica.

🎯 Propósito Educativo

Esta aplicación está diseñada como una herramienta pedagógica integral para el aprendizaje progresivo de la estadística descriptiva, siguiendo un orden temático estructurado que facilita la comprensión conceptual y práctica.

📚 Orden Temático y Contenidos

1. Aprendo a manejar la calculadora

Propósito: Preparación técnica para el manejo eficiente de calculadoras científicas Casio.

Ventaja educativa: Reduce errores de cálculo y aumenta la confianza en procedimientos matemáticos.

2. Ingreso de Datos

Propósito: Aprender la correcta captura y validación de datos estadísticos.

Ventaja educativa: Desarrolla habilidades de organización y preparación de datos para análisis.

3. Tablas de Frecuencia

Propósito: Comparar métodos de agrupación de datos (simple vs. agrupada).

Ventaja educativa: Comprende cuándo y cómo usar diferentes tipos de distribuciones.

4. Estadísticas Básicas

Propósito: Calcular e interpretar medidas de tendencia central y dispersión.

Ventaja educativa: Base fundamental para todos los análisis estadísticos posteriores.

5. Cuartiles y Outliers

Propósito: Identificar la posición relativa de datos y valores atípicos.

Ventaja educativa: Desarrolla capacidad de análisis crítico de datos.

6-10. Visualizaciones Gráficas

Propósito: Representar datos mediante histogramas, diagramas de caja, gráficas combinadas y polígonos.

Ventaja educativa: Facilita la interpretación visual y comunicación de resultados.

11-12. Tutoriales Paso a Paso

Propósito: Detallar procedimientos de cálculo manual para cuartiles y gráficos circulares.

Ventaja educativa: Refuerza el aprendizaje conceptual y metodológico.

13. Teoría de Dispersión

Propósito: Profundizar en medidas de variabilidad y coeficiente de variación.

Ventaja educativa: Conecta teoría con aplicaciones prácticas.

14. Simulación del Río

Propósito: Aplicar conceptos estadísticos en un contexto real y práctico.

Ventaja educativa: Demuestra la utilidad de la estadística en la toma de decisiones.

15. Análisis Avanzado de Outliers

Propósito: Técnicas avanzadas de detección y clasificación de valores atípicos.

Ventaja educativa: Prepara para análisis estadístico más sofisticado.
